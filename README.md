# Akıllı Müşteri Yönetim ve Analiz Sistemi

## Proje Hakkında

Bu proje, Python Programlama dersi dönem ödevi kapsamında geliştirilmiştir. Projenin amacı telekomünikasyon müşterilerinin abonelik verilerini analiz etmek, müşteri segmentasyonu yapmak ve müşteri kaybı (Churn) risklerini değerlendirmektir.

## Kullanılan Teknolojiler

* Python
* Google Colab
* Random
* Math
* Datetime

## Özellikler

* Müşteri profili oluşturma
* Otomatik müşteri ID üretme
* VIP müşteri tespiti
* Sadakat analizi
* Churn (müşteri kaybı) analizi
* Fatura hesaplama
* Hizmet analizi
* Rapor oluşturma

## Proje İçeriği

### 1. Hafta

* Veri yapıları
* Listeler
* Sözlükler
* Koşullu ifadeler

### 2. Hafta

* Fonksiyonlar
* Döngüler
* Kütüphane kullanımı
* Hata yönetimi
* Set veri yapısı

## Çalıştırma

Google Colab veya Python ortamında notebook dosyasını açıp hücreleri sırasıyla çalıştırmanız yeterlidir.
